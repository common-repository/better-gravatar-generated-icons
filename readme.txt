Bored by Identicon and MonsterId? Here is a way to add new automatic generated avatars, such as Flathash or Unicorns or funny robots

== Description ==

This plugin will add a way to use new avatars in your site from different 3d party avatar generators services, included:
- Flat hash avatars from http://flathash.com/
- Robot Hash avatars from https://robohash.org/
- Unicorns avatars from https://unicornify.appspot.com/
